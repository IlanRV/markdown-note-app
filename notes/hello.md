# Hello, World!

Welcome to the Markdown Note App. This is a sample note to demonstrate how
plain `.md` files are automatically converted to clean, readable HTML pages.

You can add as many markdown files as you like to the `notes/` folder and they
will appear on the home page instantly — no build step required.

## Why Markdown?

Markdown is simple, portable, and version-control friendly. It lets you focus
on writing without worrying about formatting tools or rich-text editors.

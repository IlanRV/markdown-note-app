# Release Plan
Category: Product
Tags: roadmap, launch, checklist

The release plan tracks the work required to turn the note app into a polished internal knowledge base. The current demo focuses on read-only notes, search, API routes, and clear rendering paths.

## Milestones

- Build modular server-rendered pages.
- Add JSON endpoints for notes, stats, and search.
- Group notes by category.
- Keep all content stored as markdown.
- Improve extraction quality with explicit route and service boundaries.

## Launch Checklist

- [x] Home page renders all notes.
- [x] Individual note pages render markdown.
- [x] API endpoints expose note data.
- [x] Search returns scored results.
- [ ] Add editing and persistence later.
